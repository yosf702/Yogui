# YOGui Desktop 🌌
The next-gen, lightweight Linux Desktop Environment & Window Manager.

## Features:
- 🚀 **Extreme Lightweight:** Designed for ARM (rasberry pi-Termux), AMD64 (Laptops-pcs), and even 32-bit legacy hardware.
- 🎨 **One-Click Dots:** Integrated Dotfiles manager (No more Git commands for themes).
- 🔗 **Dual-Engine:** Seamless support for Wayland and X11.
- 📱 **Mobile First:** Optimized for Linux-on-Android (Termux-X11).

## Supported Architectures:
- ✅ x86_64 (AMD64)
- ✅ AArch64 (ARM64)
- ✅ i386 (32-bit)
